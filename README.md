A simple weather app to help you plan your day.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AstroCaleb/weathercurve.com.git
   ```
2. Install dependencies and run locally:
   ```bash
   yarn install && yarn dev
   ```

## Usage

Once running locally, you can search any location by ZIP code or city name, or use your device's current location. Weather Curve shows current conditions, an hourly forecast you can scroll through, a minutely precipitation chart, and a full set of weather stats — feels like, humidity, wind, dew point, cloud cover, visibility, and UV index. Weather alerts are surfaced when active.

Toggle between °F and °C, and 12 or 24 hour time, at any time. All preferences and saved locations are stored in the browser's localStorage — nothing is sent to a server, and nothing is tied to Personally Identifiable Information (PII).
