# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works with multi-package repos, or single-package repos to help you version and publish your code. You can find the full documentation for it [in the readme](https://github.com/changesets/changesets)

Each changeset is a folder or a Markdown file that holds two bits of information:

1. A version type (following [semver](https://semver.org/)), and
2. Information that can go into a changelog entry for the change.

[Click here to learn what changesets are, and how to add one](https://github.com/changesets/changesets/blob/main/docs/adding-a-changeset.md).
